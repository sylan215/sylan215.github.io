Java 三大特性之封装

发表于 2017-03-06 | 分类于 programming ， java | 热度 ℃

学习Java，必须得了解Java的三大特性，今天就说说三大特性之一的封装。

封装就是把属性私有化，然后提供公共的方法来访问私有属性。
